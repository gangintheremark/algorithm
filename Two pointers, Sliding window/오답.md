연속된 자연수의 합(two pointers)  </br>
연속부분수열이랑 코드는 같지만 Time줄이기 위해서 m = n / 2 + 1 로 해결 </br>
ex) 합이 15가 되려는 연속된 자연수 중 7+8 -> 8 이 최대 (15/2+1 = 8)
```java
public int solution(int n) {
        int answer = 0, sum = 0, lt = 0;
        int m = n / 2 + 1;
        int[] arr = new int[m];
        for (int i = 0; i < m; i++)
            arr[i] = i + 1;

        for (int rt = 0; rt < m; rt++) {
           //...
        }
        return answer;
    }
```
